# Bill-o-Bill Landing Page

A high-conversion, modern landing page for Bill-o-Bill - AI-Powered Accounting Automation Platform for Indian Compliance.

## 🚀 Features

- ✅ **Fully Responsive** - Mobile, tablet, and desktop optimized
- ✅ **Modern Design** - Clean white + green theme
- ✅ **Smooth Animations** - GSAP + Lenis smooth scrolling
- ✅ **High Conversion** - Multiple CTAs, social proof, urgency elements
- ✅ **SEO Optimized** - Semantic HTML, meta tags
- ✅ **Fast Loading** - Optimized assets, minimal dependencies

## 🎨 Design Elements

- **Color Palette:**
  - Primary: Emerald Green (#10b981)
  - Secondary: Dark Green (#059669)
  - Background: White (#ffffff) + Light Gray (#f8fafc)
  
- **Typography:** Plus Jakarta Sans (Modern, professional)
- **Animations:** GSAP scroll animations, smooth scrolling with Lenis
- **Icons:** Font Awesome 6

## 🔧 Customization

### Change Colors
Edit CSS variables in `css/styles.css`:
```css
:root {
    --primary: #10b981;     /* Change primary color */
    --primary-dark: #059669; /* Change dark variant */
}
```

### Update Content
- Edit text directly in `index.html`
- Update stats in hero section
- Modify feature cards
- Change CTA buttons

### Add New Sections
1. Add HTML section in `index.html`
2. Add styles in `css/styles.css`
3. Add animations in `js/main.js`
